## Requirements

SSWebRTC is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

## Installation

```ruby
pod 'SSWebRTC'
```

## Usage

### OBJC

```objc
#import <WebRTC/WebRTC.h>
```

### SWIFT

```swift
import WebRTC
```
